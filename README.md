click this button

## Config

Enviroments list

* SLACK_TEAM_NAME
  * Your slack team name
* SLACK_TOKEN
  * Your slack api token. get from https://api.slack.com/web#auth
* SESSION_SECRET_KEY
  * For encrypt session data. first make random text and use it.
* SLACK_TEAM_DESC
  * Description of your slack
